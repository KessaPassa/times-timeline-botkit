#Summary
slack botの便利機能スクリプト  
<br>


## 1. Timeline(分報)
・`#times_{your_name}` で呟いた会話が `#times_timeline` に流れる  
・twitterや公開自分用DMみたいなもの  
<br>

## 2. Memo(メモ)
・簡易的なメモを取れる  
・チャンネル毎に保存される  

botにメンションで  
`add {message}` でメモ追加  
`remove {num}` で番号のメモを削除   
`list` で一覧表示  
<br>


## 3. Room(在室管理)
・誰が居るのかを手動で管理する  
・午前5時で強制ログアウト  
 
botにメンションで  
`login` でログイン  
`logout` でログアウト  
`room` で在室一覧  
<br>
